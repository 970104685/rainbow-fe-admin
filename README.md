# RAINBOW-FE-ADMIN 彩虹管理台

### 提交格式

- feat: 新特性
- fix: 修复问题
- refactor: 代码重构
- docs: 文档修改
- style: 代码格式修改,不是 css 修改
- test: 测试用例修改
- chroe: 其他修改, 比如构建流程,依赖管理
- ci: 持续集成相关
- anno: 代码注释相关
- perf: 性能优化
- revert: 撤销上一次提交
